# UGAHacks 11 — Visual Style Guide

This document defines the **official visual identity** for UGAHacks 11.  
All AI systems generating websites, components, graphics, or UI for UGAHacks 11 **must follow this guide exactly** unless explicitly instructed otherwise.

---

## 1. Brand Vibe & Theme

**Theme:** Magical · Celestial · Playful · Night-Sky  
**Tone:** Welcoming, whimsical, slightly mysterious, but still professional  
**Avoid:** Corporate minimalism, harsh contrast, neon cyberpunk, aggressive gradients

The design should feel like:

- A calm night sky filled with stars ✨
- Cozy, friendly magic (not dark fantasy)
- Student-driven, creative, and fun

---

## 2. Color Palette (Official)

These are the **only approved brand colors**.  
AI-generated designs should primarily use the purple gradient background with gold accents.

### Core Colors

| Name            | Hex       | Usage                                             |
| --------------- | --------- | ------------------------------------------------- |
| Mystic Purple   | `#9B6798` | Primary background (top gradients, hero sections) |
| Deep Indigo     | `#3E4C8A` | Secondary background, nav bars, footers           |
| Soft Periwinkle | `#A7B7D9` | Section backgrounds, cards, highlights            |
| Sage Green      | `#7B9A7B` | Secondary accents, subtle UI states               |
| Warm Brown      | `#A16C2A` | Rare accent (icons, dividers)                     |
| Star Gold       | `#FFCF59` | Primary accent (stars, highlights, buttons)       |
| Cream           | `#FAF3E0` | Text on dark backgrounds, soft surfaces           |

### Color Rules

- **Purple → Indigo gradients** are preferred over flat colors
- Gold (`#FFCF59`) should be used sparingly for emphasis
- Never use pure white (`#FFFFFF`) for large text blocks
- Avoid pure black; use dark purples/blues instead

---

## 3. Typography

### Primary Display Font

**Amarante**

- Used for:
  - Logos
  - Major headings (H1, H2)
  - Hero text
- Characteristics:
  - Decorative
  - Rounded
  - Magical personality

```css
font-family: "Amarante", serif;
```
